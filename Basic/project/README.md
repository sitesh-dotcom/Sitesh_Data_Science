# Sitesh Data Science & Market Analysis

A collection of Python tools for financial market analysis and data visualization.

## Key Features
* **NSE Dashboard**: Real-time tracking of market indices.
* **Market Analyzer**: Automated stock data visualization using `yfinance`.
* **Heatmaps**: Correlation analysis of stock price movements.

## Setup Instructions
1. Activate environment: `source ds_env/bin/activate`
2. Install dependencies: `pip install -r requirements.txt`
3. Launch app: `streamlit run app.py`
